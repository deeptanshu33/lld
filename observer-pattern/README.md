# Low-Level Design (LLD)

This repository documents my journey of learning and practicing **Low-Level Design (LLD)** through hands-on implementation of common design patterns and object-oriented design problems.

The goal is not just to understand the theory, but to build each pattern from scratch, understand the trade-offs, and write clean, maintainable code.

## Repository Structure

Each directory represents a separate LLD concept or design pattern.

```text
.
├── observer-pattern/
├── strategy-pattern/
├── factory-pattern/
├── builder-pattern/
└── ...
```
